Variables de Estado del Juego
Las siguientes variables son cruciales porque contienen la información fundamental para saber dónde quedó la partida.

turno: Esta variable define de qué jugador es el turno actual ("Azul" o "Rojo"). Es el punto de partida para saber quién debe jugar.

fichasaustralia y fichassumatra: Estas variables son objetos que contienen la cantidad de fichas y el color del dueño para cada país. Para reconstruir el tablero, necesitas los valores de cantidad y color de cada país.

paises: Este array de objetos es la estructura de datos más importante de tu juego. Contiene el estado de todos tus países, incluyendo el continente, fichas, colorfichas, y limitrofes. Aunque los limitrofes no cambian durante el juego, guardarlos facilita la reconstrucción.
